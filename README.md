# Documentation, Please
"Documentation, Please" is a companion app for the game "Papers, Please." In it, you can view information about game days, countries, cities, specific missions, and cumulative regulations.

"Documentation, Please" is - Nowi -'s first project for Android that moves away from local security, a simple project. It doesn't have to be so intense.

At - Nowi -, we are continuing to update "NowiVault." Thank you for your trust. 🙂
